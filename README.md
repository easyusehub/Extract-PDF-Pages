# 🗂️ Extract PDF Pages | EasyUseHub

**Extract PDF Pages** is a free online tool that lets you instantly select and extract specific pages from your PDF file. No uploads, no API, and 100% privacy — everything runs directly in your browser.

## ✨ Features
- ✅ Extract specific pages from PDF files
- ✅ Live PDF page preview with thumbnails
- ✅ Select / Deselect / Select All pages
- ✅ Fast and secure client-side processing
- ✅ No API or server upload required
- ✅ Max file size: 30MB
- ✅ Clean, premium & mobile-friendly UI

## 🚀 How to Use
1. Upload your PDF file (max 30MB)  
2. View all pages with live preview  
3. Select the pages you want to extract  
4. Click **Extract & Download PDF**  
5. Download your new PDF instantly
